Architecture Data Flow (How it's Calculated)

1. Stored Procedure Loop: distribute_carbon_credits() scans all registered RENEWABLE_ASSETS and calculates carbon credits (capacity_kw * 0.75).
2. Metric Injection: The values are physically inserted into the SUSTAINABILITY_METRICS table as immutable rows.
3. Materialized View: MV_STATE_SUSTAINABILITY runs incredibly fast SUM() and AVG() aggregation across thousands of rows, which is then fetched via the Node.js API to populate the top cards instantly!
